## Run Locally

Clone the project

```bash
  git clone git@github.com:ishantbhurani/chat-app-css-illustration.git
```

Go to the project directory

```bash
  cd chat-app-css-illustration
```

Run `index.html`

```bash
  <browsername> index.html
```

E.g.

```bash
  firefox index.html
```

```bash
  google-chrome index.html
```
